# ClawPilot

**AI Co-Founder Team for Solopreneurs**

Three AI Lobsters. One Founder. A Full Team.

## OpenClaw Hackathon 2026

ClawPilot deploys 3 AI Lobster Agents running 24/7, covering health management, operations automation, and product research. They coordinate autonomously through shared memory + event-driven architecture.

### Three Lobsters

- **CareClaw** — Health Guardian (Apple Watch integration, sleep/HR/HRV analysis, burnout detection)
- **OpsClaw** — Operations Autopilot (4-platform hotspot tracking, content generation, YouTube matching)
- **GuardClaw** — Security & Research (pain point tracking, V7 pipeline, competitor analysis)
